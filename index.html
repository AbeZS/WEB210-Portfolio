Vicious Cycles - Raleigh's All Things BMX

I was tasked with creating a website/online store for a fictional company called "Vicious Cycles," a bike store in my local area. I designed the layouts, font choices, colors, etc., in Figma. I created the logo using Adobe Illustrator/Affinity Designer. The website itself is almost purely HTML/CSS except for some Javascript for the dropdown menu. I consider this project a work in progress since I'm still working out some kinks with the new products page.
